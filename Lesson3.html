A computer system is an integrated set of components that work together to process data and perform tasks. It mainly consists of hardware and software, with various components coordinating to execute instructions.

üíΩ Hardware

Hardware refers to the physical parts of a computer that you can see and touch.

Examples: keyboard, monitor, hard disk, motherboard, CPU, RAM.

  • Input devices: allow users to send data into the computer (e.g., keyboard, mouse).
  • Output devices: display results from the computer (e.g., monitor, printer).
  • Storage devices: store data and programs permanently (e.g., hard drives, SSDs).

üíæ Software

Software is the programs and instructions that tell the hardware what to do.

  • System software: manages the computer hardware and provides a platform for applications.
    Example: Operating System (Windows, macOS, Linux)
  • Application software: performs specific tasks for the user.
    Example: Microsoft Word, web browsers, video games

üß† Memory

Memory is where the computer stores data and instructions temporarily or permanently.

  • Primary memory (RAM): temporary storage that the CPU uses for active tasks.
  • Secondary memory (Hard drives, SSDs): long-term storage for programs and data.
  • Cache memory: very fast memory inside or near the CPU to speed up processing.

üß© CPU (Central Processing Unit) & Machine Cycle

The CPU is the brain of the computer. It executes instructions and processes data.

The CPU works in a cycle called the Machine Cycle, which includes four main steps:

  • Fetch: retrieves an instruction from memory.
  • Decode: interprets the instruction.
  • Execute: performs the instruction (calculations, data movement).
  • Store/Writeback: saves the result back to memory.

üéõÔ∏è Control Unit (CU)

The Control Unit is part of the CPU. Its main job is to direct the flow of data between CPU components, memory, and input/output devices.

  • It fetches instructions from memory.
  • Decodes instructions to tell the CPU what operation to perform.
  • Coordinates other units like the ALU, memory, and I/O devices.

‚ûó Arithmetic Logic Unit (ALU)

The ALU is another part of the CPU that performs all mathematical and logical operations.

  • Arithmetic operations: addition, subtraction, multiplication, division.
  • Logical operations: comparisons like equal to, greater than, less than.

The ALU works closely with the CU, which tells it what operation to perform.
